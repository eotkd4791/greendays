### 🔑 코딩 컨벤션

- 파일명
  - `.js`, `.css` ⇒ **camelCase**
  - `.vue`(SFC) ⇒ **PascalCase**
- `변수명 / 함수명` ⇒ **camelCase**
- `컴포넌트명` ⇒ **PascalCase**
- `사용자 정의 이벤트명` ⇒ **lower-case-kebab-case**
- `Store`
  - `state` ⇒ **camelCase**
  - `getters` ⇒ **camelCase**
  - `mutations` ⇒ **UPPER_CASE_SNAKE_CASE**
  - `actions` ⇒ **UPPER_CASE_SNAKE_CASE**

### 📌 프로젝트 설명

쇼핑몰의 상품 이미지, 브랜드명, 상품명 등은 Faker.js를 이용한 더미 데이터입니다.
또한, 서버 없이 로컬스토리지로 작동하는 웹사이트입니다.
